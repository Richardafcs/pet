# ADR 模板

ADR = Architecture Decision Record。每个重要技术选择都应记录原因和后果。

## 文件命名

```text
docs/development/adr/0001-use-vite-react-typescript.md
```

## 模板

```md
# ADR-0001: 标题

## 状态

Proposed | Accepted | Superseded

## 日期

YYYY-MM-DD

## 背景

我们需要做什么决定？有哪些约束？

## 决策

我们选择什么方案？

## 备选方案

- 方案 A
- 方案 B
- 方案 C

## 理由

为什么选择这个方案？

## 后果

正面影响：

- ...

负面影响：

- ...

## 后续行动

- ...
```

## 建议首批 ADR

- 前端框架选择。
- 本地优先存储选择。
- OCR mock-first 策略。
- 是否引入后端。
- 部署平台选择。
