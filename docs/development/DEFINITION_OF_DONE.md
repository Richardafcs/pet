# Definition of Done

任务完成必须同时满足以下条件。

## 通用完成条件

- 实现符合对应需求或任务描述。
- Typecheck 通过。
- Lint 通过。
- 相关测试通过。
- 有错误状态。
- 有空状态或无数据状态。
- 文档已更新。
- 没有新增敏感日志。

## UI 功能完成条件

- 桌面和移动端可用。
- 键盘可访问主要操作。
- 风险状态不只靠颜色。
- 表单有 label、错误信息和确认反馈。
- 用户可以撤销、编辑或修正关键识别结果。

## 领域逻辑完成条件

- 核心逻辑是 pure function，除非有明确原因。
- 边界条件有测试。
- 日期相关测试传入固定 today，不依赖系统时间。
- 所有输出可解释。

## 存储完成条件

- 数据刷新后仍存在。
- 写入失败有错误处理。
- 清除数据可用。
- schema 变化有 migration。

## OCR/AI 完成条件

- 结果进入候选态。
- 用户确认前不影响库存和计划。
- 每个字段有置信度或明确说明无法判断。
- 失败时可手动添加。

## 发布完成条件

- 需求追踪矩阵状态已更新。
- 演示路径可重复执行。
- 部署检查清单通过。
- 如果真实 OCR 开启，隐私提示和密钥保护已验证。
